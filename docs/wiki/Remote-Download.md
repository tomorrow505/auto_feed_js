# Remote Download

## 功能
将当前详情页的种子推送到远程下载器（qB / Transmission / Deluge）。

## 显示条件
- 必须在设置里启用远程侧边栏。
- 必须是可准确绑定当前种子的详情页。
- 对 `torrentid` 站点（PTP/GPW/RED/OPS/DIC）会严格校验当前 URL 与解析结果一致。
- 无法准确定位种子时，侧边栏不会显示（防误推送）。

## 配置项
- 客户端地址、用户名、密码（Deluge 仅密码）
- 逻辑路径标签（label -> save path）
- 默认跳检、推送前确认、侧边栏透明度

## 测试建议
先在设置页点“测试连接”，再做真实推送验证。
