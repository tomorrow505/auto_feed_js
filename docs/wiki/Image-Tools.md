# Image Tools

## 当前能力
- 简介图片提取与替换
- 缩略图 -> 原图链接转换（按常见图床规则）
- 重传图床：PTPImg / Pixhost / Freeimage / Gifyu
- 图床桥接：hdbits / imgbox / pixhost / hdbimg / hostik

## 使用入口
- 在转发嵌入区点击图床按钮。
- 或在设置页填写图床 API Key 后使用重传。

## 行为说明
- 优先使用你在页面中手工勾选/缓存过的截图。
- 若无手工选择，则从简介图片自动提取。
- 处理结果会回写到缓存数据，供目标站预填复用。

## 常见问题
- 上传失败先检查对应 API Key。
- 某些站点图床有防盗链，先做原图 URL 转换再重传。
