# Image Tools

重构版把图片相关能力分成两类：描述文本处理，以及“图床页一键拉取”桥接。

## 描述文本处理
入口在转发面板内：
- `原图`：尽量把描述里图片转换成原图链接形式
- `PTPIMG/PIXHOST/FREEIMAGE/GIFYU`：把图片链接转存成目标图床的 BBCode
- `图片处理/Image Tools`：打开大窗口做更复杂的提取/替换/缩略等操作

## 图床页一键拉取（桥接）
当你在转发面板点击 `IMGBOX/HDBits/Hostik` 等按钮时：
1. 脚本会把解析到的截图 URL 写入临时队列（GM 存储）。
2. 打开对应图床页面后，脚本会显示“一键拉取”按钮。
3. 点击“一键拉取”会下载截图并预填到图床页面的文件选择框。

约束：
- 脚本不会自动点击“Start Upload/上传”，避免触发误操作。
