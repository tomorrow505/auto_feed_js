# Usage

本页描述重构版的“转发主链路”和常用入口，尽量贴近旧版交互但不承诺已完全对齐。

## 🚀 使用教程 (Usage Tutorial)

本教程将引导你快速掌握 Auto-Feed Refactor 的核心操作。

---

## 1. 安装与初始化
1. **安装环境**: 确保你已安装 [Tampermonkey](https://www.tampermonkey.net/) 插件。
2. **加载脚本**: 安装编译后的 `dist/auto_feed.user.js`。
3. **打开面板**: 在任意支持的站点页面，按下快捷键 `Alt + S` 弹出主设置面板。
4. **配置语言**: 在面板标题栏右侧，选择 **中文** 或 **English**。

---

## 2. 基础转载流程 (Basic Forwarding)
这是脚本最核心的功能，只需以下四步：

1. **源站准备**: 进入任何支持的种子详情页（如 M-Team, PTP, HDB）。
2. **开启面板**: 点击种子标题旁的 `转发/Reupload` 按钮，弹出转发面板。
3. **元数据增强 (可选)**: 
   - 点击 **"点击获取"** 按钮，系统将根据 IMDb ID 自动从豆瓣或 Ptgen 获取中文标题、简介与海报并存入缓存。
4. **跳转目标站**: 在面板的“常用站点”或“所有站点”中，点击你想发布的站点（如 BHD）。
5. **自动预填**: 目标站上传页打开后，脚本会自动检测缓存并填充。你会看到右下角弹出 **"Found Data & Injected"** 的提示。

---

## 3. 图像工具的使用 (Image Tools)
在转发面板或设置页面中，你可以使用强大的图像处理能力：

- **单图转存**: 粘贴一个包含防盗链的图片 URL，选择 PTPIMG 或 PIXhost，脚本会通过后端桥接自动上传并返回 BBCode。
- **批量转存**: 脚本会自动识别页面上的截图。勾选需要的图片，一键批量上传到 PIXhost，并生成带缩略图的 BBCode。
- **图床桥接**: 点击图床图标（如 HDBits），脚本会跳转并预填文件 URL，你只需手动点击“上传”即可。

---

## 4. 远程推送 (Remote Download)
1. **配置界面**: 进入设置面板 -> **远程推送** 选项卡。
2. **添加客户端**: 选择类型（qB/TR/Deluge），输入 URL 与凭据。
3. **一键推送**: 在源站种子页面的侧边栏或按钮区，点击对应的下载器图标，种子将直接推送到你的服务器。

---

## 5. 常见问题 (FAQ)
- **快捷键冲突?**: 可以在设置中修改 `Alt + S` 为其他键位。
- **预填失败?**: 请检查目标站是否已登录，并确认该站点是否在 [支持清单](Site-Support.md) 中。
- **数据不一致?**: 部分外站强制要求原版英文信息，请在转发面板中切换“发布/查重”模式。

> 想要了解更多进阶技巧？请查看 [图像工具集](Image-Tools.md) 或 [远程推送详解](Remote-Download.md)。
用于从外部数据源补齐简介/副标题等信息（按设置选择 ptgen 与 IMDb->豆瓣方式）。

## 快速搜索
- 豆瓣/IMDb 页面：标题附近插入“快速搜索”链接条。
- 站点列表页：部分站点（如 PTP/HDB）在列表行内增加“快速搜索”链接条（可在设置里开关）。

- 转发面板内的“图床”按钮组支持：
  - `原图`：把描述里的缩略图/防盗链链接尽量转为原图链接
  - `PTPIMG/PIXHOST/FREEIMAGE/GIFYU`：把图片链接转存到公共图床（需要对应 API Key）
  - `IMGBOX/HDBits/Hostik`：跳转到对应图床页面，并在图床页提供“一键拉取”把图片文件预填到 file input（不自动点击上传）
- `图片处理/Image Tools`：打开图片处理大窗口（提取/替换/缩略/链接处理等）
